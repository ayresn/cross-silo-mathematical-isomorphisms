## 3. CORE MATHEMATICAL PARALLELISM
In fluid dynamics, the Navier-Stokes equations govern the motion of an incompressible fluid:
```math
\frac{\partial \mathbf{u}}{\partial t} + (\mathbf{u} \cdot \nabla) \mathbf{u} = -\frac{1}{\rho} \nabla p + \nu \nabla^2 \mathbf{u}
```
where \(\mathbf{u}\) is the fluid velocity, \(t\) is time, \(\rho\) is the fluid density, \(p\) is the pressure, and \(\nu\) is the kinematic viscosity.

In network flow optimization, linear programming formulations are used to optimize flow through a network:
```math
\text{Minimize } \sum_{i,j} c_{ij} f_{ij}
```
subject to
```math
\sum_j f_{ij} - \sum_j f_{ji} = 0 \quad \forall i
```
```math
0 \leq f_{ij} \leq u_{ij} \quad \forall (i,j)
```
where \(f_{ij}\) is the flow rate between nodes \(i\) and \(j\), \(c_{ij}\) is the cost per unit flow, and \(u_{ij}\) is the capacity of the link between nodes \(i\) and \(j\).

These equations map onto each other through a latent space transformation where the fluid velocity \(\mathbf{u}\) corresponds to the network flow rate \(f_{ij}\), and the pressure \(p\) corresponds to the node potential.
